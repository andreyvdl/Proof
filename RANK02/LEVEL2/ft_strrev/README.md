# EN-US

> Assignment name  : ft_strrev   
> Expected files   : ft_strrev.c   
> Allowed functions:   
> 
> --------------------------------------------------------------------------------   
> 
> Write a function that reverses (in-place) a string.   
> 
> It must return its parameter.   
> 
> Your function must be declared as follows:   
> 
> `char	*ft_strrev(char *str);`
